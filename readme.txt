Программа для получения статей из группы https://vk.com/@yvkurse.

Для запуска данного скрипта в случае если у вас:

а) Linux, MacOS:
    Открыть терминал, перейти в папку со скриптом и запустить его, прописав в терминал
        "python3 vk.py"

б) Windows:
    В папке dist запустить vk.exe. Дождаться выполнения файла.
В папке появится файл yvkurse.csv, содержащий текст постов и URL.

P.S скрипт можно использовать для другой группы ВКонтакте. Для этого нужно открыть файл vk.py в текстовом редакторе
и в функции take_posts, в поле domain изменить на ссылку группы. К примеру, https://vk.com/memeboizz, в domain
вписываем memeboizz и запускаем по инструкции выше.
